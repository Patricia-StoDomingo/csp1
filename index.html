About Me...

I love designing simple and efficient websites that are functional but doesn't sacrifice the aesthetics. I am new to web development and there are still a lot of things that I am learning but I am very passionate in what I do. I used to work as an architect and I love translating my architectural design knowledge of form, utility, and beauty into code and into something that people can see and interact with. I love growth and learning and I believe that every project is an opportunity to know myself more and define the way I design.

Here are some of my works!

Food Business Website | Burger Revolution

This is a practice website I made using my knowledge on CSS styling and flexbox. This project was meant to provide a website that features a menu for the buyers to choose from and know more about Burger Revolution, its creator and its mission in providing good quality and out of the box burger ideas.

Capstone Project 1

For our first capstone project we were tasked to create our web developer portfolio. This project is my first take on Bootstrap and how to use it along with an external CSS styling. The design was primarily aiming to be minimalist, as I go along the process of getting to know how to use the framework to make a responsive website.

Capstone Project 2 & 3

This is a furniture ecommerce website project that is designed with React for frontend with integration to a backend API designed using MongoDB NodeJS and ExpressJS.
